# 日本語ファイル検索

プロジェクト内で日本語（ひらがな・カタカナ・漢字）を含むファイルを検索し、ファイルパスを一覧表示してください。

## 手順

1. Grep ツールで `[\p{Hiragana}\p{Katakana}\p{Han}]` パターンを使い、プロジェクトルート配下を検索する
2. `.git/` や `node_modules/` などは除外する
3. 結果をファイルパスのみの一覧で出力する（ファイル内容は表示しない）
